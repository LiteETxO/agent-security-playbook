What's inside

Five battle-tested components — open, customize, deploy.

The 40-Page Playbook
Deep coverage of all 7 agent threats — prompt injection, excessive agency, credential theft, data exfiltration, supply chain, insecure output execution, and identity spoofing. Real scenarios. Concrete fixes. Plain English throughout. playbook.md
4 Copy-Paste System Prompt Templates
Ready-to-deploy security templates for the four most common agent types: email agents, research/browsing agents, customer support agents, and file system agents. Each template includes injection guards, permission boundaries, and handling rules for suspicious inputs. templates/
25-Point Pre-Flight Checklist
A structured checklist covering permissions, secrets handling, input validation, output review, monitoring, and incident readiness. Run this before going live with any agent. Every checkbox has a one-sentence explanation of why it matters. checklists/preflight-security-checklist.md
Incident Response Runbook
Step-by-step guide for when something goes wrong: how to detect a compromised agent, immediate containment steps, damage assessment, recovery, post-incident review template, and notification guidance. Read this before you need it. runbooks/incident-response-runbook.md
Red Team Test Suite
5 exact prompt injection tests you can run on your own agents today. Each test includes the exact prompt to send, what a vulnerable agent does, what a secure agent does, and how to fix it if you fail. Tests cover: direct injection, indirect injection, credential extraction, excessive agency, and data exfiltration. red-team/test-suite.md
Do This First

Quick start

Three things to do in the next hour.

  • 1
    Read the Quick Start section of the playbook — it's the first 10 actions, ranked by impact. If you do nothing else today, do those 10 things. Open playbook.md and scroll to "Quick Start: 10 Things To Do Today."
  • 2
    Pick the right template for your most critical agent and paste it into your system prompt. Email agent → templates/email-agent-security.md. Research agent → templates/research-agent-security.md. Support agent → templates/support-agent-security.md. File agent → templates/file-agent-security.md.
  • 3
    Run the Red Team tests on your updated agent before you deploy. All 5 tests should pass. If any fail, the test file tells you exactly how to fix it. Tests are in red-team/test-suite.md.
